1. Create a keyspace called *killrvideo* and switch to that keyspace
CREATE KEYSPACE killrvideo
WITH REPLICATION = {
  'class':'SimpleStrategy',
  'replication_factor':1
};

USE killrvideo;

2. Create table videos
CREATE TABLE videos (
	video_id timeuuid PRIMARY KEY,
	added_date timestamp,
	description text, 
	title text,
	user_id uuid,
	primary key(video_id) 
);

3. Load the data from the videos.csv file
a) copy to docker-container: docker cp .\data\1\videos.csv cassandra-node-1:/ 
b) import data to videos table: COPY videos FROM 'videos.csv' WITH HEADER=true; (header=true - skips the first line in the file)

COPY does not require column names when the target table schema and source csv file columns match respectively.

4. Run queries:
a) SELECT * FROM videos LIMIT 10;
b) SELECT count(*) FROM videos;
